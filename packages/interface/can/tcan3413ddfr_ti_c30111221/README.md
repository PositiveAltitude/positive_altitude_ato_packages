# tcan3413ddfr_ti_c30111221

Reusable `TCAN3413DDFR` CAN FD transceiver package for 3.3 V designs.

- TI `TCAN3413DDFR`, LCSC `C30111221`
- Built-in `CAN` bus and `CAN_TTL` MCU-side interfaces
- Local 100 nF decoupling on both `VCC` and `VIO`
- 120 ohm Yageo termination routed through an open solder jumper footprint
- `STB` exposed with a weak pulldown for normal-mode default behavior
