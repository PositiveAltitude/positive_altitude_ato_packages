# mcp2518fdt_e_sl_microchip

Reusable `MCP2518FDT-E/SL` SPI to CAN FD controller package for 3.3 V Raspberry Pi designs.

- Microchip `MCP2518FDT-E/SL`, LCSC `C626758`
- Built-in `SPI` host interface and `CAN_TTL` controller-to-transceiver interface
- Local Murata `100 nF` and `1 uF` supply decoupling
- Local 40 MHz crystal with Murata `15 pF` C0G load capacitors
- Exposes `nCS`, `nINT`, `nINT0/GPIO0/XSTBY`, `nINT1/GPIO1`, and `CLKO/SOF`
